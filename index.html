From Curiosity to Creation

My journey into tech started with a single question: "How does this actually work?" That spark turned into late nights, open browser tabs, and broken layouts that slowly — beautifully — came together. Curiosity became obsession. Obsession became craft.

Building with Purpose

As a final-year IT student and active freelancer, I don't just write code — I craft solutions. Every project is a conversation between logic and design. I've learned that the best interfaces are the ones people feel without noticing.
